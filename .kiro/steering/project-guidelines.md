# Multilingual Mandi - Project Guidelines

## Project Vision
Create an inclusive agricultural marketplace that breaks language barriers for Indian farmers, enabling them to negotiate crop prices in their native languages.

## Key Features
- Multilingual support (Hindi, Punjabi, English)
- Real-time market price data
- AI-powered price negotiation
- Mobile-friendly interface
- Cultural sensitivity in design

## Technical Approach
- Use Streamlit for rapid prototyping
- Implement rule-based AI for negotiation (expandable to LLM)
- Support multiple Indian languages
- Simulate APMC market data

## Target Impact
- Digital inclusion for non-English speaking farmers
- Transparent pricing to reduce exploitation
- Faster deal negotiations
- Accessible technology for rural areas

## Development Standards
- Keep code simple and maintainable
- Focus on user experience
- Ensure cultural appropriateness
- Test with multiple languages
