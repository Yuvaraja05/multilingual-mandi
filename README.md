## Overview
A multilingual agricultural marketplace platform that bridges language barriers in rural commerce, enabling farmers to negotiate crop prices in their native languages (Hindi, Punjabi, English).

## Features
- **Multilingual Support**: Hindi, Punjabi, and English interfaces
- **Real-time Market Rates**: Live APMC market price data
- **AI-Powered Negotiation**: Smart price negotiation system
- **Mobile-Friendly**: Responsive design for rural accessibility
- **Cultural Sensitivity**: Localized greetings and terminology

## Technology Stack
- **Frontend**: Streamlit
- **Backend**: Python
- **AI/ML**: Rule-based negotiation engine (expandable to LLM)


## Usage
1. Select your preferred language from the sidebar
2. Enter the crop name you want to sell
3. View current market rates and trends
4. Negotiate prices with the AI system
5. Confirm deals in your native language
